# FastPHP

## 简述

**FastPHP**是一款简单的PHP MVC框架，主要目的是方便在 **[歪麦博客]** 上学习PHP MVC框架的同学下载源代码。
关于《手把手编写自己的PHP MVC框架》的教程，请参考网站：http://www.awaimai.com/128.html。

## 目录说明

因为默认情况下git无法提交空目录，所以在教程中提到的目录结构：

- application
- config
- fastphp
- public
- runtime
  - cache
  - logs
  - sessions
- scripts

这么多目录中，从代码仓里面clone后是没有public、runtime和scripts目录的，
但是不影响使用。
